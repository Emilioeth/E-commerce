  ## Usage/Installation
  You can install this project by cloning it onto your machine, then you can run npm i at the root folder of the project, then you will need to create a .env file with your mysql credentials, and you will also need to log in to the mysql shell and running source db/schema.sql, also optionally you can run npm run seed to seed the databases with values.

  After installing as instructed you can just run the server with npm run start and start using the database, (check the routes folder for the routes on how to add delete and update the database)
